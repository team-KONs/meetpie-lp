# MeetingFlow LP

## プロジェクト概要
MeetingFlowのランディングページ。Next.js 15 (App Router) + Tailwind CSS + Framer Motion で構築。

## 技術スタック
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## コマンド
- `npm run dev` — 開発サーバー起動
- `npm run build` — プロダクションビルド
- `npm run lint` — ESLint 実行
